# 🤖 AWS AI/ML 서비스 마스터 치트키

## 🎯 1단계: 한 눈에 보는 AI/ML 서비스 핵심 키워드 매칭

시험 문제에 아래 핵심 단어가 나오면 고민하지 말고 바로 해당 서비스를 고르세요!

* **이미지/비디오 + 객체 찾기 + 유명인 + 콘텐츠 조정 + 경로 추적** $\rightarrow$ `Amazon Rekognition`
* **음성을 텍스트로 변환(ASR) + 통화 기록 자막 + PII 마스킹** $\rightarrow$ `Amazon Transcribe`
* **텍스트를 생생한 음성으로 변환 + Lexicon + SSML** $\rightarrow$ `Amazon Polly`
* **해외 유저 콘텐츠 로컬라이제이션 + 대량 텍스트 번역** $\rightarrow$ `Amazon Translate`
* **Alexa 기술 + 대화형 챗봇 / 콜센터 봇 구축** $\rightarrow$ `Amazon Lex`
* **가상 고객 센터 + 응대 흐름 생성 + 80% 저렴한 전화 솔루션** $\rightarrow$ `Amazon Connect`
* **자연어 처리(NLP) + 이메일 감정 분석(긍정/부정) + 토큰화** $\rightarrow$ `Amazon Comprehend`
* **의사 소견서 / 환자 메모 분석 + PHI 감지(DetectPHI)** $\rightarrow$ `Amazon Comprehend Medical`
* **라벨링, 구축, 훈련, 조정 + 모든 ML 프로세스 한 곳에서 수행** $\rightarrow$ `Amazon SageMaker`
* **미래 수요 예측 + 비옷 판매량 계획 + 시계열 예측** $\rightarrow$ `Amazon Forecast`
* **자연어 문서 검색 + PDF/FAQ 답변 추출 + 피드백 기반 증분 학습** $\rightarrow$ `Amazon Kendra`
* **Amazon.com 기술 + 실시간 제품 추천 + 재순위화(Re-ranking)** $\rightarrow$ `Amazon Personalize`
* **스캔된 문서 + 손글씨 추출 + 양식/테이블 및 여권 데이터 추출** $\rightarrow$ `Amazon Textract`

## 📑 2단계: 서비스별 100% 디테일 분석 및 핵심 개념

### 👁️ 1. Amazon Rekognition (컴퓨터 비전)

기계 학습(ML)을 사용해 이미지와 비디오에서 시각적 정보를 분석하는 완전 관리형 서비스입니다.

* **상세 기능:** 이미지/비디오 내의 객체(사물), 사람, 텍스트, 장면을 찾아냅니다. 얼굴 분석 및 검색(비교)을 통한 사용자 확인, 이미지 내부의 인원수 계산이 가능합니다.
* **얼굴 데이터베이스:** "익숙한 얼굴"을 저장하여 자체 데이터베이스를 생성하거나, AWS가 제공하는 유명인 얼굴 데이터베이스와 비교할 수 있습니다.
* **핵심 사용 사례:** 라벨링, 콘텐츠 조정, 텍스트 탐지, 얼굴 탐지 및 분석(성별, 연령 범위, 감정 등), 유명인 인식, 경로 추적(예: 스포츠 경기 중 선수의 움직임 분석).

#### 🔞 Content Moderation (콘텐츠 조정)

* **개념:** 이미지나 비디오에서 부적절하거나 노출되면 안 되는 원치 않는 콘텐츠를 자동 감지합니다.
* **활용:** SNS, 방송 매체, 광고, 전자 상거래 환경에서 안전한 사용자 경험을 제공하고 규제를 준수하도록 돕습니다.
* **정밀 통제:** 감지할 항목에 대해 최소 신뢰 임계값(Confidence Threshold)을 수동 설정할 수 있습니다.
* **인적 검토 연동:** 이미지에 플래그를 지정한 후, 기계가 판별하기 어려워 실제 사람이 다시 검토해야 하는 프로세스가 필요할 때는 Amazon Augmented AI (A2I)와 통합하여 처리합니다.

### 🎙️ 2. Amazon Transcribe & Polly (오디오 형제)

#### 📝 Amazon Transcribe (음성 $\rightarrow$ 텍스트)

음성을 오디오 파일에서 텍스트로 자동 변환해 주는 서비스입니다.

* **기술적 기반:** 자동 음성 인식(Automatic Speech Recognition, ASR) 딥 러닝 프로세스를 사용하여 빠르고 정확하게 변환합니다.
* **보안 기능 (Redaction):** 개인 식별 정보(Personally Identifiable Information, **PII**)를 자동으로 식별하고 제거(마스킹)하여 보안을 준수합니다.
* **언어 지원:** 다국어 오디오가 입력되어도 오디오 내의 언어를 자동으로 식별(자동 언어 식별)하는 기능을 지원합니다.
* **사용 사례:** 고객 서비스 통화 기록 전사(콜센터 녹음본 텍스트화), 자막 자동 생성, 미디어 자산의 메타데이터 생성을 통한 완전 검색 가능한 아카이브 시스템 구축.

#### 🔊 Amazon Polly (텍스트 $\rightarrow$ 음성)

딥 러닝을 사용하여 텍스트를 인간처럼 자연스럽고 생생한 음성으로 변환(Text-to-Speech)하는 서비스입니다. 대화형 응용 프로그램을 제작할 때 유용합니다.

* **Lexicon (발음 어휘 목록):** 단어의 발음을 사용자 정의할 수 있습니다. 스타일링된 단어나 약어의 읽는 방식을 지정하여 업로드한 후 `SynthesizeSpeech` API 작업에서 적용합니다.
* *예시 (스타일링):* `St3ph4ne` $\rightarrow$ "Stephane"으로 읽도록 지정
* *예시 (약어):* `AWS` $\rightarrow$ "Amazon Web Services"로 읽도록 지정


* **SSML (Speech Synthesis Markup Language):** 일반 텍스트에 마크업 태그를 추가하여 음성을 고도로 맞춤 설정합니다.
* *조정 가능 항목:* 특정 단어나 구절 강조, 음성학적 발음 구현, 숨쉬는 소리나 속삭이기 추가, 뉴스 진행자 발화 스타일(Newscaster style) 적용 등.

### 💬 3. Amazon Translate, Lex, Connect (대화 및 상담실 구성)

#### 🌐 Amazon Translate (자연어 번역)

자연스럽고 정확한 언어 번역을 제공하는 기계 번역 서비스입니다.

* **상세 개념:** 해외 사용자를 겨냥한 웹사이트 및 애플리케이션 콘텐츠의 로컬라이제이션(Localization)을 지원하며, 대규모 텍스트를 효율적으로 대량 번역할 수 있도록 돕습니다.

#### 🤖 Amazon Lex (대화형 봇 구축)

사용자와 대화할 수 있는 챗봇 및 콜센터 음성 봇을 구축하는 서비스로, Amazon Alexa를 구동하는 기술과 동일한 기술 기반입니다.

* **핵심 기술:** 음성을 텍스트로 변환하기 위한 자동 음성 인식(**ASR**) 기술과, 호출자(사용자)의 대화 텍스트에서 실제 의도를 파악하는 자연어 이해(**NLU**) 기술을 결합하여 동작합니다.

#### 📞 Amazon Connect (클라우드 가상 고객 센터)

전화를 받고 걸 수 있는 고성능 클라우드 기반 가상 고객 센터(콜센터) 솔루션입니다.

* **기능:** 전화를 수신하고 고객의 번호 입력이나 대화에 따라 안내를 제공하는 '고객 응대 흐름(Contact Flow)'을 시각적으로 생성할 수 있습니다.
* **통합성 및 비용:** 외부 CRM 시스템이나 다른 AWS 서비스와 쉽게 연동됩니다. 기존 전통 콜센터 구축과 달리 사전 예치금이 필요 없으며, 기존 인프라 대비 **비용을 최대 80% 저렴**하게 절감할 수 있습니다.

### 📝 4. Amazon Comprehend (자연어 처리)

#### 📊 Amazon Comprehend (일반 텍스트 분석)

자연어 처리(Natural Language Processing, NLP)를 사용하여 비구조적인 텍스트에서 가치 있는 인사이트와 관계를 도출하는 완전 관리형 서버리스 서비스입니다.

* **분석 항목:** 텍스트가 작성된 언어 이해, 주요 문구(장소, 사람, 브랜드, 이벤트 등) 추출, 텍스트의 어조가 긍정적인지 부정적인지 판별하는 **감정 분석**을 수행합니다.
* **언어학적 분석:** 토큰화(Tokenization)와 품사(Parts of Speech) 분석을 사용하여 문장 구조를 정밀 분석하며, 방대한 텍스트 파일 컬렉션을 주제별로 자동으로 정리하고 분류합니다.
* **사용 사례:** 고객의 이메일이나 문의 내역을 분석하여 긍정적/부정적 경험의 근본 원인을 추적, 식별된 주제별로 문서를 자동으로 그룹화하여 관리.

#### 🏥 Amazon Comprehend Medical (의료 전용 NLP)

의사 소견서, 퇴원 요약서, 검사 결과서, 케이스 노트 등 표준화되지 않고 자유롭게 기술된 **비구조적 의료 텍스트**에서 유용한 정보를 추출하는 특화 서비스입니다.

* **보안 및 규정:** 보호 대상 개인 건강 정보(Protected Health Information, **PHI**)를 자동으로 탐지하고 식별할 수 있는 전용 **`DetectPHI` API**를 제공합니다.
* **종합 파이프라인 연동:** 문서를 `Amazon S3`에 저장해 두고 배치로 읽거나, `Kinesis Data Firehose`를 연결해 실시간 의료 데이터를 분석할 수 있으며, `Amazon Transcribe`로 의사/환자의 대화 오디오를 텍스트로 먼저 변환한 뒤 이를 수신하여 의료 분석을 수행하는 아키텍처 구성이 가능합니다.

### 🚀 5. Amazon SageMaker & 예측/추천 도구

#### 🛠️ Amazon SageMaker (엔드투엔드 ML 플랫폼)

개발자와 데이터 과학자가 머신러닝 모델을 직접 구축, 훈련, 배포할 수 있도록 돕는 완전 관리형 플랫폼 서비스입니다.

* **상세 개념:** ML의 전 과정(데이터 라벨링, 모델 구축, 알고리즘 훈련, 하이퍼파라미터 조정, 운영 배포)을 분산된 환경에서 서버를 일일이 프로비저닝할 필요 없이 한 곳에서 모두 통합 수행할 수 있도록 지원합니다. (예: 시험 점수 예측 모델을 직접 설계하고 튜닝할 때 활용)

#### 📉 Amazon Forecast (시계열 수요 예측)

과거 시계열 데이터를 기계 학습으로 분석하여 미래의 정확한 수치를 예측하는 완전 관리형 서비스입니다.

* **특징:** 데이터의 단순 추세만 확인하는 전통적인 통계 방식보다 **최대 50% 더 정확한 예측력**을 보여주며, 예측 모델 개발 시간을 몇 달에서 몇 시간 수준으로 단축합니다.
* **사용 사례:** 비옷의 미래 판매량 예측, 제품 수요 계획, 재무 흐름 계획, 인프라 자원 공급 계획 등.

#### 🛒 Amazon Personalize (실시간 맞춤화 추천)

사용자 개인에게 최적화된 맞춤형 추천 기능을 애플리케이션에 즉시 구현해 주는 완전 관리형 머신러닝 서비스입니다.

* **기술적 배경:** **Amazon.com** 실제 쇼핑몰 웹사이트에서 수십 년간 검증된 추천 엔진과 동일한 기술을 기반으로 작동합니다.
* **기능:** 맞춤형 제품 추천, 품목 재순위화(Re-ranking), 맞춤화된 직접 마케팅 세팅이 가능합니다. (예: 유저가 원예 도구를 집중 구매했다면 관련 상품을 최우선으로 배치)
* **장점:** 복잡한 기계학습 솔루션을 밑바닥부터 구축·학습·배포할 필요 없이, 기존 웹사이트, 모바일 앱, SMS/이메일 마케팅 시스템에 API 연결만으로 **며칠 만에 구현**이 가능합니다. (소재 및 소매점, 미디어, 엔터테인먼트 산업에 이상적)

### 📄 6. Amazon Kendra & Textract (문서 탐색 및 추출)

#### 🔍 Amazon Kendra (지능형 검색 엔진)

머신 러닝 기술이 탑재되어 사용자가 질문을 던지면 문서 내부에서 정확한 정답을 찾아내 주는 완전 관리형 문서 검색 서비스입니다.

* **대상 문서:** Text, PDF, HTML, PowerPoint, MS Word, FAQ 등 다양한 포맷의 문서 파일에서 지식을 추출합니다.
* **자연어 검색:** 키워드 매칭이 아닌 "휴가 신청은 어떻게 하나요?"와 같은 자연어 형태의 질문을 완벽히 이해합니다.
* **증분 학습 (Incremental Learning):** 사용자의 상호 작용 및 실제 검색 결과 클릭 피드백을 시스템이 지속적으로 학습하여, 시간이 흐를수록 검색 결과의 선호도와 정확도를 스스로 높여갑니다.
* **세부 튜닝:** 데이터의 비즈니스적 중요도, 문서의 신선도(최신성), 또는 관리자가 지정한 사용자 정의 필터를 기반으로 검색 결과 상위 노출 순위를 수동으로 세부 조정(Boosting)할 수 있습니다.

#### 🖨️ Amazon Textract (스캔 문서 데이터 추출)

AI와 ML 기술을 사용하여 스캔된 이미지나 문서 파일에서 텍스트, 손글씨, 데이터를 자동으로 정밀 추출하는 서비스입니다.

* **정밀 기능:** 단순 글자 읽기(OCR)를 넘어 문장 안의 **양식(Form)과 테이블(Table) 구조를 인식**하여 키-값 쌍으로 데이터를 원형 그대로 추출할 수 있습니다. PDF나 일반 이미지 등 어떤 유형의 문서든 입력 처리가 가능합니다.
* **산업별 사용 사례:**
* *금융 서비스:* 인보이스 영수증 처리, 재무 보고서 데이터 추출.
* *의료 분야:* 복잡한 의료 기록 및 병원 보험 청구서 전산화.
* *공공 부문:* 손으로 작성된 세금 신고 양식, 신분증 및 여권 정보 자동 스캔 및 입력 시스템.

노트에 명시된 디테일들을 융합하여 시험이나 실무에 반드시 출제되는 **종합 AI 비즈니스 워크플로우** 구조입니다. 눈으로 흐름을 읽어두세요!

### 시나리오: 글로벌 의료/고객 콜센터 자동화 및 문서 자산화 시스템

$$\begin{aligned}
& \text{1. 고객 통화 음성 수신} \longrightarrow [\text{Amazon Connect (가상 콜센터)}] \\
& \text{2. 음성 파일 전사 및 가공} \longrightarrow [\text{Amazon Transcribe (ASR / PII 마스킹)}] \\
& \text{3. 다국어 자막/텍스트 생성} \longrightarrow [\text{Amazon Translate (글로벌 로컬라이제이션)}] \\
& \text{4. 의료 및 일반 텍스트 분석} \longrightarrow [\text{Amazon Comprehend / Medical (감정 분석 및 PHI 감지)}] \\
& \text{5. 스캔된 처방전/의료 기록 결합} \longrightarrow [\text{Amazon Textract (손글씨 및 양식 테이블 추출)}] \\
& \text{6. 최종 지식 창고 저장 및 탐색} \longrightarrow [\text{Amazon S3 데이터 레이크}] \longleftarrow [\text{Amazon Kendra (자연어 피드백 검색)}]
\end{aligned}$$

* **아키텍처 가치:** 전화를 `Connect`로 받아 `Transcribe`로 글자를 파낸 후, 개인 정보(`PII`)는 숨깁니다. 외국어는 `Translate`로 번역하고 `Comprehend`로 고객 화남/기쁨의 감정을 분석합니다. 병원 소견서나 처방전은 `Textract`가 양식 그대로 긁어모아 `S3`에 적재하며, 이 모든 방대한 지식은 직원들이 `Kendra`를 통해 자연어로 검색해 피드백을 주며 증분 학습을 시키는 완전한 서버리스 AI 파이프라인이 완성됩니다.
